Data Engineering Project 1 - AC Transit

Senior Data Engineer – Clever File Processing Project

One of the data engineering projects I'm particularly proud of is the Clever File Processing Project for AC Transit. I took complete end-to-end ownership of this project across all phases. The main challenge was the large, inconsistent datasets spread across multiple formats, making ingestion and transformation complex.

  • Used Azure Synapse Analytics to orchestrate ingestion from CSV, APIs, and SQL databases into ADLS.
  • Leveraged PySpark to clean, standardize, and enrich the files for downstream consumption.
  • Implemented Apache Airflow for scheduling and monitoring reliable pipelines.
  • Built Python-based visualizations to validate processed data and share operational insights.

This automated solution improved data quality, eliminated manual intervention, and enabled AC Transit to process large and complex datasets at scale.
